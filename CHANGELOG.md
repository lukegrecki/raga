## [0.1.0] - 2026-04-28

### Added
- Initial release
- `raga lookup` — look up a raga by name with detailed information
- `raga list` — list all ragas with filters (thaat, time, mood, season)
- `raga suggest` — get raga suggestions based on current time or specified filters
- `raga play` — play arohana and avarohana via FluidSynth with customizable Sa reference and tempo
- `tala lookup` — look up a tala by name with detailed information
- `tala list` — list all talas with filters (beats, feel, tempo)
- `tala suggest` — get tala suggestions based on filters
- Shell completions for bash, zsh, fish, and PowerShell
- Audio playback via FluidSynth with configurable SoundFont support
- RAGA_SOUNDFONT environment variable for SoundFont path configuration
